Color Modes in Bootstrap

Setting the color scheme in Bootstrap (as of 5.3) is super easy. You can either add data-bs-theme="dark|light" to the html attibute:

<html lang="en" data-bs-theme="dark">

You can either set the color-scheme on the :root element for the full page or on sections or individual elements. You can use media queries to target.

:root {
	color-scheme: dark light;
}
				
@media (prefers-color-scheme: dark) {
	.example.dark {
		background-color: #4b2e83;

		h3, p {
			color: #e8e3d3;
		}
	}
}

@media (prefers-color-scheme: light) {
	.example.light {
		background-color: #e8e3d3;

		h3, p {
			color: #4b2e83;
		}
	}
}
